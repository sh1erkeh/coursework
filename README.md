## Руководство по запуску
- Установите `cmake`:
```bash
sudo apt install cmake (для ubuntu/debian)
```
- Перейдите в папку с проектом
- В терминале последовательно введите команды:
```bash
cmake -S build/ && cd build/
make
./timetable
```
